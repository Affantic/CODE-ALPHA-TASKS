# Crankshaft Design in SolidWorks

## Overview
This project is a 3D crankshaft model designed and rendered in :contentReference[oaicite:0]{index=0}.  
The model was created as part of mechanical design practice to improve CAD modeling, feature creation, and rendering skills.

---

## Features Used
- Revolve Boss/Base
- Extruded Boss
- Fillet
- Chamfer
- Circular geometry creation
- Material appearance and rendering

---

## Design Description
The crankshaft consists of:
- Main shaft sections
- Offset crank webs
- Bearing journals
- Output shaft end

The geometry was modeled with dimensional accuracy and smooth transitions between features to resemble a real mechanical component.

---

## Objectives
- Practice mechanical component modeling
- Improve SolidWorks workflow efficiency
- Learn shaft and crank geometry creation
- Enhance rendering and presentation skills
